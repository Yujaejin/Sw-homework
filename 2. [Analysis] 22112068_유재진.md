## **1. Introduction**

### **Usefulness-**
&ensp;Focus Tracker는 사용자가 집중 시간과 휴식 시간을 직접 설정하고 이를 반복적으로 관리할 수 있도록 하여 효율적인 시간 관리를 가능하게 한다. 또한 세션 기록과 통계 기능을 통해 자신의 집중 패턴을 확인할 수 있어 자기주도 학습을 지원하는 데 유용하다.
<br/>
<br/>
### **Significance-**
&ensp;본 시스템은 단순한 타이머 기능을 넘어 사용자 데이터를 기반으로 분석 및 시각화를 제공하고, 전체 사용자 랭킹을 통해 동기 부여와 경쟁 요소를 도입했다는 점에서 의의가 있다. 이를 통해 사용자의 집중력 향상과 지속적인 학습 습관 형성에 기여할 수 있다.
<br/>
<br/>
### **Expandability-**
&ensp;Focus Tracker는 사용자 데이터와 서버 기반 구조를 활용하여 다양한 기능 확장이 가능하다. 예를 들어, AI 기반 집중 패턴 분석, 친구 간 랭킹 비교, 목표 설정 기능, 모바일 앱 연동 등으로 확장할 수 있으며, 사용자 증가에도 대응할 수 있도록 시스템 확장이 용이하다.

<br/>

## **3. Domain analysis**  

### &ensp;1) Control

&emsp;Focus Tracker 시스템에서 공통적으로 사용되는 기능들을 포함한 클래스이다.  

ex) 화면 전환, 메시지 출력, 팝업창 표시...  

<br/>

### &ensp;2) User

&emsp;시스템을 사용하는 사용자 정보를 저장하는 클래스이다.  

ex) 이메일, 비밀번호, 닉네임, 누적 집중 시간...  

 <br/>

### &ensp;3) Login

&emsp;사용자의 로그인 및 인증 기능을 담당하는 클래스이다.  

ex) 로그인 확인, 비밀번호 검증...  

 <br/>

### &ensp;4) SignUp

&emsp;사용자의 회원가입 기능을 담당하는 클래스이다.  

ex) 회원 정보 입력, 계정 생성...  

 <br/>

### &ensp;5) Timer

&emsp;포모도로 타이머 기능을 수행하는 핵심 클래스이다.  

ex) 타이머 시작, 일시정지, 재개, 종료...  

 <br/>

### &ensp;6) TimerSettings

&emsp;사용자가 설정한 집중 시간 및 휴식 시간 정보를 저장하는 클래스이다.  

ex) Focus Time, Break Time...  

 <br/>

### &ensp;7) SessionData

&emsp;사용자의 집중 세션 기록 정보를 저장하는 클래스이다.  

ex) 시작 시간, 종료 시간, 세션 길이, 완료 여부...  

 <br/>

### &ensp;8) SessionTracking

&emsp;사용자의 세션 데이터를 기록하고 관리하는 클래스이다.  

ex) 세션 저장, 세션 조회...  

 <br/>

### &ensp;9) Statistics

&emsp;사용자의 세션 데이터를 분석하여 통계를 생성하는 클래스이다.  

ex) 총 집중 시간 계산, 평균 집중 시간 계산...  

 <br/>

### &ensp;10) PersonalStatistics

&emsp;사용자의 개인 통계 데이터를 관리하는 클래스이다.  

ex) 일일 집중 시간, 누적 세션 수...  

 <br/>

### &ensp;11) GlobalRanking

&emsp;전체 사용자 데이터를 기반으로 랭킹을 계산하는 클래스이다.  

ex) 사용자 정렬, 순위 계산...  

 <br/>

### &ensp;12) Notification

&emsp;집중 시간 종료 및 휴식 시간 시작 알림을 처리하는 클래스이다.  

ex) 알림 출력, 알림 메시지 생성...  

 <br/>

### &ensp;13) MainScreen

&emsp;사용자가 메인 화면에서 수행하는 기능들을 담당하는 클래스이다.  

ex) 타이머 화면 표시, 메뉴 이동...  

 <br/>

### &ensp;14) DatabaseManager

&emsp;데이터베이스와의 연결 및 데이터 저장·조회 기능을 담당하는 클래스이다.  

ex) 사용자 데이터 저장, 세션 기록 조회...  

 <br/>

### &ensp;15) ServerManager

&emsp;서버와의 통신 및 요청 처리를 담당하는 클래스이다.  

ex) 사용자 인증 요청, 랭킹 데이터 요청...
