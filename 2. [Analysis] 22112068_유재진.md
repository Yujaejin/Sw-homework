## **1. Introduction**

### **Usefulness-**
Focus Tracker는 사용자가 집중 시간과 휴식 시간을 직접 설정하고 이를 반복적으로 관리할 수 있도록 하여 효율적인 시간 관리를 가능하게 한다. 또한 세션 기록과 통계 기능을 통해 자신의 집중 패턴을 확인할 수 있어 자기주도 학습을 지원하는 데 유용하다.  
<br/>
<br/>

### **Significance-**
본 시스템은 단순한 타이머 기능을 넘어 사용자 데이터를 기반으로 분석 및 시각화를 제공하고, 전체 사용자 랭킹을 통해 동기 부여와 경쟁 요소를 도입했다는 점에서 의의가 있다. 이를 통해 사용자의 집중력 향상과 지속적인 학습 습관 형성에 기여할 수 있다.
<br/>
<br/>

### **Expandability-**
Focus Tracker는 사용자 데이터와 서버 기반 구조를 활용하여 다양한 기능 확장이 가능하다. 예를 들어, AI 기반 집중 패턴 분석, 친구 간 랭킹 비교, 목표 설정 기능, 모바일 앱 연동 등으로 확장할 수 있으며, 사용자 증가에도 대응할 수 있도록 시스템 확장이 용이하다.
