## **1. Introduction**

&ensp;현대의 대학생들은 학업, 과제, 시험 준비 등 다양한 활동을 수행해야 하지만 스마트폰, SNS, 영상 플랫폼 등 여러 외부 요인으로 인해 집중력을 유지하는 데 어려움을 겪는 경우가 많다. 특히 본 프로젝트의 개발자는 3년 만에 복학한 이후 학습에 다시 적응하는 과정에서 장시간 집중하는 것이 쉽지 않다는 점을 체감하였다. 이러한 문제는 학습 효율 저하와 시간 관리의 어려움으로 이어질 수 있다.

Focus Tracker는 이러한 문제를 해결하기 위해 포모도로 기법(Pomodoro Technique)을 기반으로 설계된 집중 시간 관리 애플리케이션이다. 사용자는 집중 시간(Focus Time)과 휴식 시간(Break Time)을 직접 설정할 수 있으며, 타이머 기능을 통해 체계적으로 학습 시간을 관리할 수 있다. 또한 집중 세션 기록을 데이터로 저장하고 분석하여 개인 통계를 제공하며, 전체 사용자 랭킹 기능을 통해 동기 부여와 경쟁 요소를 제공한다.

이를 통해 사용자는 자신의 학습 패턴을 객관적으로 확인할 수 있으며, 지속적인 집중 습관 형성과 생산성 향상을 기대할 수 있다.

&ensp;이 문서는 Focus Tracker를 개발하기 위한 세 번째 단계인 Design에 대한 내용으로, 실제 구현을 위한 시스템 구조와 클래스 설계, 객체 간 상호작용, 상태 변화 및 사용자 인터페이스를 다이어그램을 통해 시각적으로 표현한다.

## **2. Class diagram**  
### 2.1. User

#### &ensp;1. Attributes
&emsp;name:String : 사용자 이름  

&emsp;ID:String : 사용자 DI  

&emsp;tel:String : 사용자 전화번호  

&emsp;email:String : 사용자 E-Mail  

&emsp;approach:int : 사용자에게 부여되는 권한  


#### &ensp;2. Methods
&emsp;+login(email, password) : 로그인 수행  

&emsp;+logout() : 로그아웃 수행  

&emsp;+getProfile() : 사용자 정보 조회  

<br/>

### 2.2. TimerSettings

#### &ensp;1. Attributes
&emsp;focusTime : 집중 시간  

&emsp;breakTime : 휴식 시간  

#### &ensp;2. Methods
&emsp;+setFocusTime(time) : 집중 시간 설정  

&emsp;+setBreakTime(time) : 휴식 시간 설정  

&emsp;+loadSettings() : 저장된 설정 불러오기  

<br/>

### 2.3. Timer

#### &ensp;1. Attributes  
&emsp;currentTime : 현재 남은 시간  

&emsp;isRunning : 타이머 동작 여부  

#### &ensp;2. Methods
&emsp;+start() : 타이머 시작  

&emsp;+pause() : 타이머 일시정지  

&emsp;+resume() : 타이머 재개  

&emsp;+stop() : 타이머 종료  

&emsp;+reset() : 타이머 초기화  

<br/>

### 2.4. SessionData

#### &ensp;1. Attributes  
&emsp;startTime : 세션 시작 시간  

&emsp;endTime : 세션 종료 시간  

&emsp;duration : 세션 지속 시간  

&emsp;completed : 세션 완료 여부  


#### &ensp;2. Methods
&emsp;+saveSession() : 세션 기록 저장  

&emsp;+getSessionInfo() : 세션 정보 조회  

<br/>

### 2.5. SessionTracking

#### &ensp;1. Attributes  
&emsp;SessionTracking  

#### &ensp;2. Methods
&emsp;+addSession(session) : 세션 추가  

&emsp;+getSessions() : 세션 목록 조회  

&emsp;+deleteSession() : 세션 삭제  

<br/>

### 2.6. Statistics

#### &ensp;1. Attributes  
&emsp;totalSessions : 총 세션 수  

&emsp;totalFocusTime : 총 집중 시간   

#### &ensp;2. Methods
&emsp;+calculateStatistics() : 통계 계산  

&emsp;+getTotalFocusTime() : 총 집중 시간 반환  

&emsp;+getAverageFocusTime() : 평균 집중 시간 반환  

<br/>

### 2.7. PersonalStatistics

#### &ensp;1. Attributes  
&emsp;dailyFocusTime : 일일 집중 시간  

&emsp;weeklyFocusTime : 주간 집중 시간  

#### &ensp;2. Methods
&emsp;+getDailyStats() : 일일 통계 조회  

&emsp;+getWeeklyStats() : 주간 통계 조회  

<br/>

### 2.8. GlobalRanking

#### &ensp;1. Attributes  
&emsp;rankingList : 사용자 랭킹 목록  

#### &ensp;2. Methods
&emsp;+calculateRanking() : 랭킹 계산  

&emsp;+getRanking() : 전체 랭킹 조회  

<br/>

### 2.9. Notification  

#### &ensp;1. Attributes  
&emsp;message : 알림 메시지  

#### &ensp;2. Methods
&emsp;+sendNotification() : 알림 전송  

&emsp;+createMessage() : 알림 메시지 생성  

<br/>

### 2.10. DatabaseManager

#### &ensp;1. Attributes  
&emsp;connectionStatus : DB 연결 상태  

#### &ensp;2. Methods
&emsp;+saveUser() : 사용자 정보 저장  

&emsp;+loadUser() : 사용자 정보 조회  

&emsp;+saveSession() : 세션 데이터 저장  

&emsp;+loadSession() : 세션 데이터 조회  

<br/>

### 2.11. ServerManager

#### &ensp;1. Attributes  
&emsp;serverStatus : 서버 연결 상태  

#### &ensp;2. Methods
&emsp;+authenticate() : 사용자 인증 수행  

&emsp;+requestStatistics() : 통계 요청  

&emsp;+requestRanking() : 랭킹 요청  

&emsp;+syncData() : 서버와 데이터 동기화
